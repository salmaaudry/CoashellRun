Thank you for downloading.

-Modals folder contains RedCrownedCraneModel.fbx.
This file contains only the mesh data.
-Animations  folder contains RedCrownedCraneAnimations1.fbx.
This file contains only the animations data.
-Scenes folder contains a demo scene.
In this demo scene, friction of the physic material of the ground is set 0.
-Prefabs folder contains a prefab that was used in the demo scene.
