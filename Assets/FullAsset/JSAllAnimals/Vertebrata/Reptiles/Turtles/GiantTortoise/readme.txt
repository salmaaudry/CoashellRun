Thank you for downloading GiantTortoise.

This is a realistic 3d model and animations of a GiantTortoise. 
-Models folder contains 6 FBX files of giant tortoise models.
-Textures folder contains textures.
-Animations folder contains FBX files of animations.
Walk animations are compatible with root motion.
-Animations/OnPlace folder contains FBX files of on place variations of walk animations.
These are not used in the demo scene.

These are main contents of this package.

Following is an example use of GiantTortoise. 
-Scenes folder contains demo scene.
-Prefabs folder contains prefabs that are used in the demo scene.
-Scripts folder contains scripts that are used in the demo scene.
GiantTortoiseRandomWalk is a simple random walk script.
GiantTortoiseUserController is a User controller as a third person character.

-Animator folder contains animator controller that is used the prefab.
