Thank you for downloading JS all animals pack.

This package contains 3d model and animations of animals.
This package contains
5 invertebrate(Ammonite, Anomalocaris,Ceratocanthus, Shrimp, Snowcrab)
2 amphibians(FrogAndSkeleton, PoisonDartFrog)
13 birds(Archaeopteryx, BaldEagle, BarnOwl, BowerBird, Cassowary, HarpyEagle, HummingBird, Penguin, Pigeon, RedCrownedCrane, Swan, Ostrich, Secretarybird)
6 fishes(Dunkleosteus, Sardine, Seahorse, Angelfish, Arowana, Pirarucu)
3 mammals(Capybara,Maned wolf, Orca).
12 reptiles(2 dinosaurus(AllosaurusAndSkeleton, Gallimimus), Mosasaurus, 3 lizards(Gecko, Iguana, Komododragon),6 turtles
(AlligatorSnappingTurtle, BoxTurtle, RedfootedTurtoise, SeaTurtle, AfricanSpurredTortoise, Giant tortoise)).
